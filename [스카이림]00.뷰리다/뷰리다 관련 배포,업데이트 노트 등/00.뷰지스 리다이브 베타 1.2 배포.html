새로시작 방법



단축키

마우스 좌클릭 = 평타 / (활 꺼내고 있을 때) 활 시위 당기기

V = 강공 / (활 꺼내고 있을 때) 특수 공격키

마우스 우클릭 = 방어 / (활 꺼내고 있을 때) 속사

SHIFT = (발도 중) 닷지 / (납도 중, 누르는 동안) 걷기 / (활 시위 당기고 있는 도중) 회피 사격

숫자키 1 = 손에 장작한 무기/마법 해제

숫자키 2, 3, 4, 5 = 순서대로 내추럴, 하이, 미드, 로우 스탠스 변경

CAPS LOCK = 휠러 호출

Q = 용언/능력 사용

E = 상호작용

N = 흥분도(OSL) 확인

M = 지도

Z  = 즐겨찾기

C = 자동이동 / 3인칭 카메라 어깨 변경

H = 소유 중인 말과 상호작용 (가방 열기, 부르기/대기 시키기) / (길게 눌렀을 때) 소유 중이 말 근처로 소환

T = 대기

I = 인벤토리

J = 퀘스트 저널

K = 오르가즘 저항/장비 벗기 (구속구)

L = 최근 구속구 탈출 시도 / (길게 눌렀을 때) 구속구 목록 보기 

백스페이스 = 파티 위젯 표기 변경

[ = 파티 위젯 상태창 켜기/끄기

,< = 파티 위젯 켜기/끄기

.> = 소환수 위젯 켜키/끄기



F5 = 저장 (빠른저장 아님, 수동 저장과 동일)

F7 = 오바디 체형 설정

F8 = 상황 별 표정 변화 정지/재생 토글

F10 = 아이템창고

F11 = 커뮤니티 쉐이더 성능 모니터링

F12 = 커뮤니티 쉐이더 설정

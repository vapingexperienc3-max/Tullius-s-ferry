🔍스림정보 업스케일러 키면 화면 1/4만 나오면서 잘리는 버그, 캐릭터가 사선으로 걸어가는 버그해결법이 안올라와있어서 아는거 다 올림
모찌슬릿 
추천 1 비추천 0 댓글 1 조회수 948 작성일 2025-07-23 20:26:33 수정일 2025-07-23 20:32:32
https://arca.live/b/tullius/143139902


일단 위 버그는 나도몰랐던 업스케일링 두배 소매넣기 이벤트에 당해버려서 의도했던 것보다 한번 더 업스케일링 되는 경우 또는 게임 셋팅이 QHD화면 설정인데 내가쓰는 모니터가 FHD인 경우임. 

 그리고 캐릭터가 공간 지각능력을 상실하고 대각선으로 걸어가는 버그가 처음엔 스무스캠같은거 오류인줄 알았는데 알고보니 화면이 1/4가 잘려서 그랬던거더라. 해결법은 동일.


1. 업스케일링 


MO2의 DLAA, DLSS 부분에서 나는 기본제공 DLAA로 셋팅했으니 이거 기준으로 말하면 

우클릭 ->탐색기 열기 -> skse ->plugin -> skyrim upscaler.ini -> 

[Settings]

mEnableUpscaler = true

#0 = DLSS, 1 = FSR2, 2 = XeSS, 3 = DLAA（equal to DLSS Native，reserved for compatibility）

mUpscaleType = 0

#0 = Performance

#1 = Balanced

#2 = Quality

#3 = UltraPerformance

#4 = UltraQuality

#5 = Native （DLSS Native = DLAA）

mQualityLevel = 2


사실 멀바껏는지 기억이안나서 일단 복붙했음. 퀄리티레벨 바꾼거같음.  대충 enable upscaler랑 DLAA랑 구조상 충돌일어나는거같음. 

(mEnableUpscaler=false로 셋팅하고 퀄리티레벨을 원래 원하던대로 해도 오류해결할수 있을거같음) 


2. 스카이림 게임 폴더 내 enbseries.ini

[GUI]

HighResolutionScaling=true를 펄스로 변경. 이또한 해로운 스케일링이다. 


3. MO2에서 sse diplay 트윅 우클릭 후 탐색창 ->skse ->플러긴 ->SSEDisplayTweaks.ini

Fullscreen = false

Borderless = true

BorderlessUpscale = true

FHD인 경우 앞에 # 지우고 

Resolution = 1920x1080

ResolutionScale = 1


낮은 확률로 AutoCenter = 펄스에서 트루로 변경 

ScalingMode = unspecified 를 센터드로 변경 이걸로 해결될수도 있음


이것저것 다뒤적거리다가 옴닉의 힘을 빌어 해결함. 정작 저 맨 위 해결법은 여기에 설명된 저 윗내용은 없는거같더라고, 내가 한 세시간정도 뒤적거려봐서 암. (세시간동안 암걸렸다는 뜻)

나도 이 게임 구조는 잘 모르고 프로그래밍도 잘 몰라서 뇌피셜 동원해서 문제될거 생각하다가 해결한 방법임. 

일단 난 셋다 수정해서 해결됏고, 사람에따라 경우의수는 다를거라 생각함. 일단 내 경우에서는 위 문제를결정적으로 해결한 방법은 1번이였음. 


태그- dlaa, dlss, 업스케일러, 1/4, 설산, 화면 잘림, 잘리는, 잘려, 메인 화면, 캐릭터가 옆으로감 대각선, 이상한 곳
